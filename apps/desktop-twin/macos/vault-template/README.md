# Coday Twin Vault

This is your Coday Twin workspace, managed by the Coday Twin desktop application.
It also serves as an Obsidian vault for your notes and knowledge management.

## Structure

- `inbox/` - Quick capture area
- `notes/` - Your notes and documents
- `projects/` - Project-specific content

## Getting Started

Open this folder in Obsidian to start using it as a vault.
The Coday Twin application will manage AI-assisted operations in this workspace.

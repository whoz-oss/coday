# Extras & Resources

This folder contains supporting files that aren't vault notes themselves.

---

## Contents

```
x/
├── Templates/      ← Note templates (see Templates/README.md)
└── Attachments/    ← Images, PDFs, other media files
```

---

## Templates

Ready-to-use note structures for common note types. See `Templates/README.md` for guidance.

## Attachments

Drop images, PDFs, and other files here when linking them from notes.

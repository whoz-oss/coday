## Getting Started

1. Install prerequisites: Node.js 22+, pnpm
2. Clone: `git clone https://github.com/whoz-oss/coday.git`
3. Install: `pnpm install`
4. Set API keys: `export ANTHROPIC_API_KEY=...` and/or `export OPENAI_API_KEY=...`
5. Run: `pnpm web:dev` (development) or `npx --yes @whoz-oss/coday-web` (published version)
6. Open http://localhost:3000 in your browser
